Projects

  • Artemis - Oberon-07 module collection for OBNC, Oxford and Ofront+ compilers.
  • pttk, a tool kit for Pandoc generated websites. It includes a proof-of-concept Pandoc Pre-processor called “prep”. “prep” will take a JSON document, transform it into to YAML front matter suitable for processing with Pandoc. It also includes “blogit” a blog site generator and manager used to prepare this website’s blog
  • obc-3-libext - exploring compatibility in Karl Landström’s Extension Library modules with Mike Spivey’s Oxford Oberon Compiler
